# 🚗 汽车产业资讯采集工具

汽车产业资讯采集工具，为科室同事提供便捷的新闻资讯采集和报告生成功能。

## 功能特性

- ✅ **8大领域采集**：行业动态、企业要闻、高层动态、零部件、新技术/新趋势、新车上市、宏观经济政策、政策法规
- ✅ **时间范围选择**：近一周/近两周/近一个月/自定义
- ✅ **多选领域**：支持单选/多选采集领域
- ✅ **Word导出**：一键生成专业资讯简报
- ✅ **定时任务**：支持每天定时自动采集
- ✅ **进度显示**：实时显示采集进度
- ✅ **系统托盘**：支持最小化到托盘运行

## 快速开始

### 方式一：直接运行exe（推荐）

1. 双击运行 `汽车资讯采集工具.exe`
2. 选择采集领域和时间范围
3. 点击「开始采集」
4. 采集完成后点击「生成Word」

### 方式二：源码运行

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 安装浏览器（可选）
python -m playwright install chromium

# 3. 运行程序
python main.py
```

### 方式三：打包exe

```bash
python build.py
```

## 注意事项

1. **首次运行**：首次运行需要下载Chromium浏览器（约150MB）
2. **网络要求**：采集过程中需要保持网络连接
3. **输出位置**：Word文档默认保存在 `E:\openclaw\workspace\auto_news_collector\output`
4. **定时任务**：设置定时任务后，程序会在后台运行，到时自动采集

## 常见问题

### Q: 采集失败怎么办？
A: 程序会自动保存缓存到 `data\cache` 目录，下次运行时可使用缓存数据。

### Q: 如何修改定时采集时间？
A: 在界面中选择「每天定时」，然后选择具体时间。

### Q: 可以只采集某些领域吗？
A: 可以，在「采集领域」中勾选需要采集的领域即可。
