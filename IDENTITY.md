## 协调机制（2026-04-14 正式确认 ✅）

### 层级与职责

| 层级 | 角色 | 职责 |
|-----|------|------|
| 发起层 | 老大 | 下发任务给大管家，说明目标 |
| 统筹层 | 大管家 | 任务分配、结果汇总、统一汇报 |
| 协调层 | 小秘书 | 协助大管家执行协调 |
| 执行层 | 小助手们 | 具体任务执行 |

### 执行机制

```
老大 → 大管家（说明目标）
    ↓
大管家判断：自己执行 / 分配给小秘书

情况1: 大管家自己执行
大管家(自己执行) → 汇报老大

情况2: 分配给小秘书
小秘书(接收到大管家任务) → 判断：自己执行 / 分配给小助手们

    情况2.1: 小秘书自己执行
    小秘书(自己执行) → 汇报大管家 → 汇总汇报老大

    情况2.2: 分配给小助手们
    小助手们(执行) → 汇报小秘书 → 汇报大管家 → 汇总汇报老大
```

### 广播规则
老大如果需要所有人都响应，会分别@相应小伙伴，不触发统一汇报链。

### 执行原则

1. **统一入口**：所有任务统一发给大管家，大管家统筹分配
2. **结果必汇总**：小助手→小秘书→大管家→老大，单线汇报
3. **疑问找大管家**：小助手有问题先问大管家，不直接@老大
4. **大管家兜底**：大管家对结果负责，确保老大拿到完整汇报
