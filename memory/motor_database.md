## 四驱数据语义拆分规则

当H列="四驱"时，需要对D/E/F列进行语义拆分：

### 拆分优先级

1. **语义标记优先**：匹配"前/xxx/后/xxx"或"前:xxx/后:xxx"格式
   - 例：`前:BMW AG/后:采埃孚` → 前值=BMW AG, 后值=采埃孚

2. **按"/"分隔符数量**：
   - **2个"/"**：直接拆分
     - 例：`Bosch/Mercedes-Benz` → 前值=Bosch, 后值=Mercedes-Benz
   - **3个"/"**：第一个为前值，其余合并为后值
     - 例：`东莞/A/B` → 前值=东莞, 后值=A/B
   - **4个"/"**：前两个合并为前值，后两个合并为后值
     - 例：`TZ220XYBZ/TZ220XYCA/TZ220XYBZ/TZ220XYCB`
     - → 前值=`TZ220XYBZ/TZ220XYCA`, 后值=`TZ220XYBZ/TZ220XYCB`
   - **5个及以上**：前两个为前值，其余为后值

3. **逗号/分号分隔**：拆分

4. **公司名称含"/"**：含"公司"、"汽车"、"动力"等关键词的公司名，不拆分，原值填入两列
