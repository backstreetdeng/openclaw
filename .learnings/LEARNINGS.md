## summarize-pro Skill 中文触发指令 (2026-04-23)

**来源：** 网上搜索 + 用户确认

**核心原理：**
- 模型识别"总结、摘要、提炼、专业总结"等中文意图
- 自动匹配并加载 summarize-pro 技能
- 执行专业摘要提取

**有效的中文触发指令（任选其一）：**
1. 帮我专业总结这段全文，提取分析摘要
2. 对下面内容做专业摘要提取，保留关键信息、数据、时间、价格、定位，语言精炼，不添加额外内容
3. 请提炼核心要点，生成结构化摘要
4. 总结这篇文章，只保留事实、数据、结论

**与英文触发词的区别：**
- 英文触发词：summarize / tldr / bullet points / key takeaways 等
- 中文触发词：总结 / 摘要 / 提炼 / 专业总结 等
- 两者都能触发skill，但中文用户更习惯使用中文指令

**应用场景：**
- Python脚本中调用 openclaw agent 时使用
- 需配合临时文件传递长文本
- 示例：请读取文件 {temp_file} 的内容，然后按照以下要求执行...

**注意事项：**
- 如果文本不超过200字，直接返回原文
- 超过200字才需要进行摘要提炼
- 临时文件使用后需自动清理
